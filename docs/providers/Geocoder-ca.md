# Geocoder.ca

Geocoder.ca - A Canadian and US location geocoder.
Using Geocoder you can retrieve Geolytica's geocoded data from Geocoder.ca.

## Python Example

```python
>>> import geocoder
>>> g = geocoder.geolytica('New York City')
>>> g.json
...
```
